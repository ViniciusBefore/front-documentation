# Instalação

  1. Crie um fork pessoal a partir deste repositório REPOSITÓRIO-HUB
  2. Clone o fork em seu ambiente local (git clone...)
  3. Após a clonagem, entre no diretório ./hubvendas-web e insira um remote correspondente ao repositório original
    git remote add hub https://github.com/beforeti/hubvendas-web.git
    OBS: Isso é para poder manter o teu local sincronizado com o repositório do HUB
  4. Duplique o arquivo .env.example e o renomeie para .env
  5. Configure seu .env para o ambiente de desenvolvimento:
    VUE_APP_ENV=development
  6. Agora é preciso configurar os acessos ao Localhost para simular o acesso de cada Provider
    - No terminal execute:
    sudo nano /etc/hosts
    - Edite o arquivo para que contenha os provedores:
    127.0.0.1 local.hubdevendas.localhost 127.0.0.1 local-alares.hubdevendas.localhost 127.0.0.1 local-neonfibra.hubdevendas.localhost
    - Salve as alterações
    // OBS: Esses são os links de acesso ao sistema

# Execução

Após a instalação, dentro do diretório do hub, execute
docker-compose up
